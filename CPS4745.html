CPS 4745                                 Visualization Design and Development                

Content: This course will focus on software design and development to represent data in visual formats from various data sources. Students will learn how to develop applications and present information from the database in understandable and effective methods in charts, graphs, dashboards and dynamic web pages. Students will do hands-on projects and compare their results with popular visualization software.

Course Student Learning Outcomes:  Upon completion of this course, the student will be able to:

A. Explain the key techniques and theory used in visualization, including data models, graphical perception and techniques for visual encoding and interaction.

B. Present information in an understandable, effective, and aesthetic manner, for the purposes of explaining ideas and analyzing data.

C. Utilize popular API and software to visualize data

D. Design visual tools that allows user to visualize and interact with data from the database.

Assessment and course grading  

The course includes an exam, self-study, presentations, and projects, a total of 1000 points.

  • 1 Midterm (250 pts)
  • 1 Study and presentation (200 pts) using software – Tableau, Google Looker, MS Power BI.
  • 1 Individual project and presentation (200 pts) using JS, Google Charts, Plot.js, HTML, PHP, MySQL
  • 1 Team project and presentation (200 pts) using JavaScript, Ajax, CSS, HTML, PHP, MySQL
  • Attendance (150 pts, 6pts/each class, must attend at least 60 minutes/each class, maximum 150 pts)

The Digital Gradebook (in Canvas) will be used to record student grades.
